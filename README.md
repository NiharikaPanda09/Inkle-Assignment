# Inkle Table Assignment

Features:
- Table built using `@tanstack/react-table` showing customers from the /taxes API
- Edit modal to change `name` and `country`, country list fetched from /countries API
- On Save, data is PUT back to the /taxes/:id endpoint

Setup:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

The app will open at the Vite dev URL (usually http://localhost:5173).
